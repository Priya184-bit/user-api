# User API 🚀

A simple RESTful API built using Node.js and Express for managing users. This project demonstrates core backend concepts such as routing, middleware, validation, and error handling.

---

## 📌 Features

* Create, Read, Update, and Delete (CRUD) users
* RESTful API design
* Middleware for logging requests
* Validation for user input
* Proper HTTP status codes
* In-memory data storage

---

## 🛠 Tech Stack

* Node.js
* Express.js
